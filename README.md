## Run Locally

Clone the project

```bash
  git clone https://github.com/lutfurrahman20/weather-app.git
```

Go to the project directory

```bash
  cd weather-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
